# Homelab-KVM-based-Infrastructure-Setup-using-Bash-Scripts

1. Clone the Git repo to your host machine
git clone https://github.com/NonsoNnamani/Homelab-KVM-based-Infrastructure-Setup-using-Bash-Scripts.git

2. Change directory into the Project Folder
cd Homelab-KVM-based-Infrastructure-Setup-using-Bash-Scripts

3. Make the bash files executable
chmod +x 01-setup-kvm-on-host-machine.sh \
         02-create-virtual-networks-for-kvm.sh \
         03-create-vm-instance.sh \
         03-provision-k8s-nodes-static-ip.sh \
         04-provision-k8s-nodes-dynamic-ip.sh
